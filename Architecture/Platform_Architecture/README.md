# Platform Architecture

## Overview

The Life Skills Bible Study Platform is a cloud-based content delivery and workflow management system built using Blogger, Google Sheets, Google Apps Script, and JavaScript.

## Core Components

- Blogger Front End
- Google Sheets Data Layer
- Google Apps Script Automation
- Source of Truth Governance
- MatchKey Routing
- Public/Private Workbook Architecture

## Design Goals

- Maintainability
- Scalability
- Governance
- Workflow Automation
- Dynamic Content Delivery
